Simple by design

Three steps to a working app

No config files to copy-paste. No tutorials to follow. Answer a few questions in plain English and you're ready to build.

1

Run one command

Nothing to install globally. One command and it walks you through everything.

npx create-vibe-project
2

Answer plain-English questions

Do you need a server? Do you need to save data? Do users need to log in? No jargon required — pick what fits your app.

3

Start building

Add your API keys to .env, then run dev to see it live. One more command pushes it to the internet.

npm install && npm run dev
